# AI-Friendly Website Audit Tool

A simple landing page that allows users to scan their website and get an AI-readiness score.

## Features

- Beautiful, modern landing page
- URL input and website scanning
- AI audit that checks for:
  - Title tags
  - Meta descriptions
  - Structured data (Schema.org)
  - Semantic HTML elements
  - Image alt text
  - Heading hierarchy
  - Open Graph tags
  - Robots meta tags
- Score display with detailed breakdown

## Installation

1. Install dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Open your browser and navigate to:
```
http://localhost:3000
```

## Usage

1. Enter a website URL in the input field
2. Click "Scan my site"
3. View your AI-readiness score and detailed audit results

## Development

For development with auto-reload:
```bash
npm run dev
```

## Technologies

- Node.js & Express (backend)
- HTML/CSS/JavaScript (frontend)
- Cheerio (HTML parsing)
- Axios (HTTP requests)
