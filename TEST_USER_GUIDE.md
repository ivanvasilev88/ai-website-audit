## How to Test

### 1. Open the Website
- Go to the URL provided above
- You'll see the AI Website Audit tool

### 2. Scan a Website
- Enter any website URL (e.g., `https://example.com`)
- Click "Scan my site"
- Wait a few seconds for analysis

### 3. View Results
- **Score**: See the AI interpretability score (0-100)
- **Free Insights**: See what AI currently understands (green boxes)
- **Locked Insights**: See blurred recommendations (need to unlock)

### 4. Test Payment Flow (Optional)
- Click "See the full AI interpretation"
- Enter your email
- Complete demo payment
- View all insights
- Download PDF report

### 5. Test Admin View
- Visit: `https://your-app-name.up.railway.app/admin`
- See all scans and scores
